## chat con Expresion Regular

Este es una aplicación donde se puede mensajear en tiempo real y en cada mensaje enviado
se detecta caracteristicas con expresiones regulares.

Ejecutar este comando para reconstruir los módulos de node y poder ejecutar el código

```
npm install
```

### Puede hacer

-Mandar mensaje simultaneamente con diferentes usuarios

- detecta en el mensaje las palabras que inician en mayuscula.
- detecta en el mensaje las palabras que no terminan en vocal.
- detecta en el mensaje la cantidad de palabras.
- detecta en el mensaje el numero de vocales.
- detecta en el mensaje los numeros presentes.
